# 术语表（Glossary）

> 统一团队与 Agent 用词，减少歧义。按字母序维护「中文（English）」。

| 术语 | English | 定义 |
|------|---------|------|
| 架构决策记录 | ADR | 记录重大架构选择及其理由的可追溯文档 |
| 规范驱动开发 | Spec-driven / Specification-Driven | 先定义规格与契约，再实现与验收 |
| 上下文工程 | Context Engineering | 为模型裁剪、组织、检索可用上下文的工程实践 |
| 交接协议 | Handoff Protocol | 多 Agent 之间传递任务状态的结构化字段约定 |
| 检查点 | Checkpoint | 长时任务可恢复的中间落盘状态 |
| 发布门禁 | Release Gate | 合并/发布前必须通过的检查集合 |
| 人机协作 | HITL / Human-in-the-loop | 高风险步骤需人工批准再继续 |
