# 待办（TODO）目录

- `workflow.md`：任务生命周期、DoD、与路线图关系

## 使用建议

- 人类团队：可继续用 Issue Tracker；每条 Issue 对应一份按 [templates/task-template.md](../../templates/task-template.md) 填写的摘要放在此目录或链到 Issue。
- 纯文档驱动：在此目录创建 `TASK-xxx.md` 文件。

## 优先级约定（示例）

| 标签 | 含义 |
|------|------|
| P0 | 阻塞生产或合规 |
| P1 | 本迭代必须完成 |
| P2 | 可排期 |
| P3 | 锦上添花 |
