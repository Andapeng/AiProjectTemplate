# 发布门禁细则（Release Gates Detail）

## 1. 门禁级别

| 级别 | 触发场景 | 要求 |
|------|----------|------|
| **L0 本地** | 开发者/Agent 日常 | 相关单元 + lint（若配置） |
| **L1 CI** | PR / 合并前 | L0 + 契约/集成子集 + 关键 Eval |
| **L2 预发** | 对外可见前 | L1 + E2E smoke + 安全扫描（若具备） |
| **L3 生产** | 正式发布 | L2 + 变更审批 + 回滚方案 |

## 2. 豁免（Exception Process）

- 必须：记录理由、批准人、补救截止时间（通常写在 ADR 附录或任务单）。
- 禁止：长期静默豁免核心契约与关键 Eval。

## 3. AI 特有风险检查

- [ ] Prompt 变更是否触达计费/合规表述？
- [ ] Tool 权限是否扩大？是否更新 `agents/` 说明？
- [ ] 新模型是否需重新标定评测基线？
